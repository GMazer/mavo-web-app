# Backend API

## Structure
- `src/app.ts`: Main entry point.
- `src/routes/`: API route definitions.

## Setup & Run

1.  **Install dependencies**:
    ```bash
    cd backend
    npm install
    ```

2.  **Development**:
    ```bash
    npm run dev
    ```
    Server runs on http://localhost:8080.

3.  **Build & Start**:
    ```bash
    npm run build
    npm start
    ```

## Endpoints

-   `GET /api/health`: Health check.
